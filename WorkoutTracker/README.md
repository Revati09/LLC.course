# Workout Tracker

A modern, fully functional workout tracking website built with pure HTML, CSS, and JavaScript.

## Features

- **Modern UI**: Colorful design with gradients, glassmorphism effects, and smooth animations
- **Responsive Design**: Works perfectly on mobile and desktop devices
- **Dark Mode**: Toggle between light and dark themes
- **Local Storage**: All data persists between sessions
- **Workout Tracking**: Add, edit, and delete workout entries
- **Progress Analytics**: View total workouts, most frequent exercises, and weekly progress
- **Personal Bests**: Track highest weights lifted for each exercise
- **Rest Day Reminders**: Get notified when you haven't worked out for a couple of days
- **Search & Filter**: Filter workouts by date and search by exercise name
- **Workout Plans**: Predefined plans for different fitness levels

## Pages

1. **Home**: Hero section with motivational quote and call-to-action
2. **Tracker**: Add new workouts and view current session workouts
3. **Plans**: Browse predefined workout plans (Beginner, Intermediate, Advanced)
4. **History**: View all past workouts with filtering and summary statistics
5. **Profile**: User profile with progress overview and personal bests

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations, Glassmorphism)
- JavaScript (ES6+, Local Storage)
- Font Awesome Icons
- Google Fonts (Poppins)

## How to Run

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Or run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then open `http://localhost:8000` in your browser

## Usage

- Navigate between pages using the top navigation bar
- Click the moon/sun icon to toggle dark mode
- On the Tracker page, fill out the form to add workouts
- Use the History page to view past workouts and filter them
- Check your progress on the Profile page

## Browser Support

Works in all modern browsers that support ES6 JavaScript and CSS Grid/Flexbox.
